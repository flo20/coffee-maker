# COFFEE MAKER

A fictitious coffee bar with a coffee machine with which you can connect via NFC and compose your coffee via this app.

### How to access app locally

### `yarn install`

### `yarn start`

## Technologies and libraries used

- React
- React Icons
- Sass
- Netlify
